Welcome to the Yatta Checkout for Eclipse demo

In this demo you can see how Yatta Checkout for Eclipse works by testing it with a demo plugin.

Yatta Checkout enables developers to sell their software tools and products directly in the Eclipse IDE, as well as offering a website integration.

For users this means easier, faster and more convenient access to the tools they need, safe and secure payments, and a central place to manage purchases and subscriptions.

1

Starting the demo

Click “Start demo” on the right.

2

Using the Checkout to purchase a demo license

The demo plugin is locked until you get a license. To get one, click “Subscribe” for a monthly or yearly license, or “Purchase” for a one-time purchase. If you do not have a Yatta account, you will be prompted to create one.

3

Completing the purchase with demo payment & billing information

The Yatta Checkout demo uses test data for payment and billing: no real transactions take place. Upon completing the purchase, you will receive a license so you can use the demo plugin immediately—without entering a license key or leaving your Eclipse IDE. Yatta Checkout takes care of that in the background.
